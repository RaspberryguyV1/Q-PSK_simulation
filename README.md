# Carrier Wave Simulation

## Overview
This repository contains a project that simulates a Q-PSK modulation and visualizes the results. 

## Repository Structure
* **C++ source code:** Responsible for running the core simulation and generating data.
* **Matlab scripts:** Responsible for the visualization of the generated carrier wave.
* **sygnal.txt file:** Acts as a simulated "cable", transferring data output from and to the C++ simulation, also allows the use of the Matlab visualization tool.

## Prerequisites
* C++ compiler (I used the Visual Studio application)
* Matlab
